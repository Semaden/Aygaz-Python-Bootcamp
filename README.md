# Taş, Kağıt, Makas, İğne ve İplik Oyunu

Bu proje, Python ile yapılmış bir Taş, Kağıt, Makas, İğne ve İplik oyununu içerir.

## Kurallar

- Taş makası kırar.
- Taş iğneyi kırar.
- Makas ipliği keser.
- Makas kağıdı keser.
- İplik taşı yener.
- İplik kağıdı sarar.
- Kağıt taşı sarar.
- Kağıt ipliği sarar.
- İğne kağıdı deler.
- İğne ipliği yener.
